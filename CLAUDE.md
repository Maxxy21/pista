## Git / commit conventions

- **NEVER** add Claude (or any AI) as a commit co-author, contributor, or collaborator. Do not append `Co-Authored-By: Claude` or any "Generated with Claude" trailer to commits or PR bodies. This is a hard rule with no exceptions.
- Keep commit messages **short and human** — a single concise lowercase subject line (e.g. `add design system spec`, `fix logo blob`). No long bodies or bullet lists unless explicitly asked.
